# Tumblr Backup
`tumblr-backup` is an easy way to backup Tumblr sites using wget.

## Usage

    $ tumblr-backup \
    $   --sites="sitea.tumblr.com siteb.tumblr.com" \
    $   --destination="/tumblr-backup" \
    $   --reduce-to-zip

## Installation

    $ brew tap jbryanscott/tumblr-backup
    $ brew install tumblr-backup
